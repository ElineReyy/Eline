# Trivial Pursuit Game Customization and Usage

## Overview
This document provides guidance on how to customize and use the Trivial Pursuit game.

## Customization
To customize the game, follow these steps:
1. **Game Pieces**: Choose different game pieces from the provided options.
2. **Questions**: Edit the question bank by modifying or adding new questions in the `questions.json` file.
3. **Game Rules**: Adjust the rules in the `rules.md` file as needed to fit your preferences.

## Usage
To play the game, follow these steps:
1. **Setup**: Gather your game pieces, and ensure you have a complete set of questions.
2. **Players**: Decide on the number of players and assign game pieces.
3. **Gameplay**: Follow the rules outlined in the `rules.md` for playing the game.

Enjoy your game!
